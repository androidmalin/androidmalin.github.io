1.安装并启动一个的APK（例如app.apk这个文件）
首先需要将aapt配置到环境变量中，aapt的路径在`/sdk/build-tools/xxxxx/`<br/>
![](http://ogxkun013.bkt.clouddn.com/aapt2.png)<br/>
a.获取APK的包名
`aapt dump badging app.apk | grep package | awk '{print $2}' | sed s/name=//g | sed s/\'//g`<br/>
例如得到如下的结果
com.malin.world<br/>
b.得到APK的启动Activity类名
`aapt dump badging app.apk | grep activity | awk '{print $2}' | sed s/name=//g | sed s/\'//g`<br/>
例如得到如下的结果
com.malin.world.activity.MainActivity<br/>
c.将APK文件安装到设备
`adb install -r app.apk`<br/>
d.使用adb命令启动应用
`adb shell am start -n package/launchable-activity`<br/>
例如：<br/>
`adb shell am start -n com.malin.world/com.malin.world.activity.MainActivity`<br/>
如下所示
![](http://ogxkun013.bkt.clouddn.com/aapt3.png)<br/>
e.将上面的四个命令写成shell脚本(例如命名为`runApp.sh`文件)
![](http://ogxkun013.bkt.clouddn.com/aaptRunApp.png)<br/>
d.给脚本授权可执行的权限。
`chmod +x runapp.sh`<br/>
f.可以将该脚本加入环境变量中。
这里可以参考这篇文章 [Linux,Unix下环境变量的配置](http://www.malin.xyz/ma/lin/update/2016/11/17/adb.html)<br/>
g.执行试试看。
![](http://ogxkun013.bkt.clouddn.com/installRun.png)<br/>
该APK被安装到了手机上并启动了<br/>
