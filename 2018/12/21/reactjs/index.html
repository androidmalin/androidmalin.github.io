Mac下React环境配置

发表于 2018-12-21 |

1.安装NodeJs

NodeJs
安装node的时候会自动安装npm

brew update
brew install node

查看nodeJs的版本

node -v
npm -v

包管理工具

查看nodeJs软件包信息
brew info node
brew info npm
brew deps node
brew deps npm

已安装的包和对应的依赖
brew deps --installed

brew leaves
show installed formulae that are not dependencies of another installed formula.

代理配置

npm config set registry https://registry.npm.taobao.org --global
npm config set disturl https://npm.taobao.org/dist --global
vim ~/.npmrc 确保内容为 registry=https://registry.npm.taobao.org

验证配置结果
npm config get registry

2.使用 create-react-app 快速构建 React 开发环境

安装工具
npm install -g create-react-app

生成项目
create-react-app my-app

进入目录
cd my-app/

启动
npm start
