Ubuntu修改Host不生效的问题

1.编辑/etc/hosts文件

sudo vim /etc/hosts

添加host，或者注释掉host

2.重启网络

sudo /etc/init.d/networking restart

或者

3.使用ping xxxxx测试修改后是否生效。

4.如果没有生效的话，有可能是因为有一次使用vim,然后异常退出，产生了hosts.swp文件。

使用如下的命令查找hosts.swp文件。

locate update

locate *.swp

5.找到后删除这个文件
sudo rm -rf /xx/xx/hosts.swp

sed ‘s/\”//g’
